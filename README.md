# PHUTURE – Frontend

PHUTURE's frontend is a modern Vue 3 e-commerce interface designed to provide a clean, responsive, and scalable shopping experience.

The frontend communicates with the PHUTURE backend API to handle authentication, product browsing, and user interactions.

⚠️ Project Status  
🚧 This project is currently under active development. New features and improvements are being added regularly.

---

# Tech Stack

## Core Framework
- Vue 3 (Composition API)
- Vite

## State Management
- Pinia

## Routing
- Vue Router

## Styling
- Tailwind CSS

## API Communication
- Axios

# Running the Frontend Locally

Install dependencies

npm install

Run development server

npm run dev

The application will run at

http://localhost:5173

---

# Environment Variables

Create a .env file

VITE_API_BASE_URL=http://localhost:5000/api

---

# Backend API

The frontend communicates with the PHUTURE backend API for:

- Authentication
- Product data
- Orders
- User information

Example API URL

http://localhost:5000/api
